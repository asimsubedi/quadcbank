# QuadCBank

## Introduction
Web application demonstrating the online bank for customer. Customer can login, check his account balance, view transactions and transfer balance.

## Technologies used

1. Front End
  - HTML
  - CSS
  - Javascript
  - Jquery
  - Bootstrap Library
2. Back End
  - Java
  - JSP
  - Spring MVC
  - Spring JDBC
3. Tools
  - Jdk 1.8
  - Eclipse
  - MySQL Workbench
  - Tomcat Server
  - Git
  - Maven
